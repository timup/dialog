About dialog

dialog was born out of the desire to make connections- linking the Triangle’s entreprenurial community to local tech services targeted at early-stage projects.

The web is a tremendous catalog of valuable information, but filtering through it can be time-prohibitive for any professional. Additionally, there's rarely one perfect solution for a project's needs.

Of course, if appropriate, vaporware would like to be part of the project's solution. If this truly is the right step, both parties will know. Otherwise, dialog exists in a space of tremendous talent and capability, and will be happy to make intelligent recomendations.

By no means does dialog aim to solve all problems. Rather, dialog is a conduit- facilitating discussion, so that a proper direction can be realized.
